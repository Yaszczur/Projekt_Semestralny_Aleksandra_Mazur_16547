# GENERATOR HASEŁ - Aleksandra Mazur 16547

## Krótki opis aplikacji
Aplikacja okienkowa wykonana w technologii WPF dzięki, której użytkownik może wygenerować bezpiecznie losowe hasła o określonej przez niego długości. Każde hasło jest zapisywane automatycznie w lokalnym pliku tekstowym, a użytkownik może szybko sprawdzić historię wygenerowanych haseł bezpośrednio w aplikacji.

## Lista funkcji aplikacji
1. **Generowanie haseł** - Generowanie haseł z użyciem losowych znaków ASCII o długości wybranej przez użytkownika
2. **Zapis haseł** - Automatyczne zapisywanie każdego nowego wygenerowanego hasła do pliku
3. **Plik z hasłami** - Wyświetlanie listy wszystkich dotychczas wygenerowanych haseł w czytelnym oknie

## Instrukcja uruchomienia aplikacji
1. Sklonuj repozytorium na swój dysk lub pobierz je jako plik (.zip)
2. Otwórz plik rozwiązania **ProjektSemestralny_Aleksandra_Mazur_16547** w programie **Visual Studio**
3. Kliknij przycisk **Uruchom bez debugowania (Ctrl + F5)** w górnej części menu Visual Studio, aby skompilować i włączyć aplikację

## Dodatkowe elementy aplikacji
1. **WPF** - Czytelny interfejs graficzny wykonany za pomocą kodu XAML
2. **OOP** - Architektura aplikacji oparta na programowaniu obiektowym. Logika została odseparowana od interfejsu i podzielona na osobne klasy
3. **IO** - Wykorzystanie przestrzeni nazw **System.IO** do trwałego zapisywania danych na dysku i dodatkowo sprawdzania istnienia pliku przed jego faktycznym odczytem
